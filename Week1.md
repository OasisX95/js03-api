# Week 1 - Review
List and its manipulation: 
```javascript
hello = [1, 2, 3, 4, 5, 6, 7];
hello[0];
//1
hello[4];
//5
hello[2] = "vaewwefawf";
hello;
//[1, 2, "vaewwefawf", 4, 5, 6, 7];
```
For loops:
```javascript
hello = [1, 2, 3, 4, 5, 6, 7];
for (let i = 0; i < hello.length; i++){
    console.log(hello[i]);
}
//1
//2
//3
//4
//5
//6
//7
```
simple string concatenation:
```javascript
a = "awd";
b = "rty";
c = a + b;
//c = "awdrty"
```
